# Final Revision VII
Retro game for christmas

Developer setup
=======

## Requirements
* NodeJS 

## Steps
1. Clone repo to your local machine \
`$ git clone https://github.com/DotDigitalAgency/final-revision.git`

2. Install Parcel  \
`$ npm install -g parcel@next`

3. Run `$ parcel index.html`
